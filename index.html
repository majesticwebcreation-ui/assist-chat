Questions teams ask before launch.

What data can I train the AI on?

You can use public URLs, uploaded documents, and raw text snippets in this static demo. A production version would replace the demo JSON with a database and ingestion API.

How do message credits work?

Each customer and AI exchange can use credits. Longer conversations consume more than single-question chats.

Can I remove branding?

Yes. The Growth and Scale plans include branding removal controls in the widget customizer.

Can I use multiple agents for different websites?

Yes. The dashboard includes a workspace switcher and sample agents for e-commerce, SaaS, and hospitality.

How does human handoff work?

The AI can flag a conversation, collect email details, and preserve transcript history for the assigned teammate.

Is there a mobile workflow or notifications?

The interface is responsive, and the demo includes notification-ready activity and coming-soon channel cards.
